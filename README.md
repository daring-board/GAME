# GAN
Generative adversarial networks

## Requirement

* Python3
* keras
* tensorflow
* matplotlib
* numpy

## usage
* dcgam_bike.py  
Put sample images on ./data/set directory.  
And input following command to prompt.
```
python dcgam_bike.py
```
Output path: ./images_dcgan

* gan_bike.py  
Put sample images on ./data/set directory.  
And input following command to prompt.
```
python gam_bike.py
```
Output path: ./images
